# Ultrafast molecular dynamics
Analysis of the redshift between absorption and fluorescence spectra of a cluster of optically excited organic molecules. Data was collected in real time on a strak camera and also simulated via hpc on GROMACS.

### Features

- Stokes' shift theory of fluorescence.
- Data analysis of the energy absorption/emission spectra for a cluster of Benzene, ChloroBenzene, DiChloroBenzene
- Ultrafast molelucar dynamics analysis and data collection via streak camera
- Simulation of a 600 molecules cluster on GROMACS and comparison between the simulated and experimental results
